# Strategic Product Placement Analysis: Unveiling Sales Impact with Tableau Visualization

## Problem Statement
The objective of this project is to analyze how product positioning affects sales performance.

## Data Collection
Dataset Source: Kaggle  
Dataset: Impact of Product Positioning on Sales

## Data Cleaning
- Checked missing values
- Verified data types
- Removed inconsistent entries

## Visualizations
- Avg Sales by Product Category
- Foot Traffic vs Sales Volume
- Competitor Price vs Price
- Promotion Impact on Sales
- Product Category vs Price

## Dashboard
An interactive Tableau dashboard was created combining all visualizations.

## Web Integration
The dashboard was embedded into a Flask web application using HTML.

## Conclusion
Tableau helps analyze the impact of product placement on sales performance.
